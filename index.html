Variables

Variables are named containers that store data values. In Python, you don't need to declare variables explicitly - just assign a value!

Key Rules:
  • Must start with letter or underscore
  • Can contain letters, numbers, underscores
  • Case-sensitive (age ≠ Age)
  • Cannot use Python keywords

Keywords

Keywords are reserved words in Python that have special meanings. You cannot use them as variable names, function names, or identifiers.

Control Flow

Functions & Classes

def return class lambda yield

Boolean & None

True False None and or not

Imports & Others

Keywords in Action

Identifiers

Identifiers are user-defined names for variables, functions, classes, modules, etc. They must follow specific naming conventions.

Valid Identifiers

  • user_name
  • _private
  • firstName
  • age2
  • PI
  • myClass

Invalid Identifiers

  • 2age (starts with number)
  • my-name (contains hyphen)
  • for (Python keyword)
  • user name (contains space)
  • @variable (special character)
  • class (Python keyword)

Comments

Comments are notes in your code that Python ignores. They help explain what your code does and make it more readable.

Single-line Comments

Use # for single-line comments

# This is a single-line comment

Multi-line Comments

Use triple quotes for multi-line comments

"""This is a multi-line comment"""

Docstrings

Special comments that describe functions/classes

def func():
    """This is a docstring"""

Syntax:

if-else Statement

The if-else statement provides an alternative path when the condition is false.

Syntax:

if condition:
    # code block if condition is True
else:
    # code block if condition is False
